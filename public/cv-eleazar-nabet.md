## Positionnement

- Développement front-end : HTML, CSS, JavaScript, TypeScript, React, Next.js, Vite, Tailwind, interfaces responsive.
- Développement back-end : Node.js, Express, FastAPI, APIs REST, SQLite, PostgreSQL, Drizzle, authentification par clé API.
- DevOps et systèmes : Linux, Shell, systemd, Git, GitHub, Docker, Tailscale, Vercel, auto-hébergement.
- IA et automatisation : LLM, Ollama, Claude provider, llama.cpp, agents IA, STT/TTS, NéronOS, Home Assistant, Telegram, Google Calendar.

## Projets sélectionnés

- neron_llm : microservice LLM FastAPI avec providers Ollama, Claude et llama.cpp, router et stratégies single/parallel/race.
- neron_vocal : interface vocale Next.js/TypeScript avec HTTPS via Tailscale, STT côté core et TTS navigateur.
- neron_doctor : service FastAPI de diagnostic, monitoring systemd/journalctl et autocorrection simple.
- neronHUB : dashboard Full Stack React/Express/PostgreSQL/Drizzle avec OpenAI SDK.
- Orane Coaching : application React/Vite avec PostgreSQL/Neon, Google Calendar API et dashboard admin.
